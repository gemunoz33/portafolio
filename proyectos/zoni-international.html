Zoni International

Sistema de gestión de reclutamiento docente internacional con arquitectura monolítica y soporte para múltiples regiones.

// El Desafío

El reto principal consistió en diseñar y centralizar un ecosistema de reclutamiento docente para una organización con presencia internacional. El cliente necesitaba eliminar la fragmentación de datos y estandarizar el proceso de selección. Se desarrolló una plataforma que automatiza desde el registro inicial del candidato y la evaluación de competencias lingüísticas mediante pruebas de nivel, hasta la gestión de entrevistas virtuales y la sincronización final con el ERP corporativo.

Requerimientos Clave

  • Sistema de registro multiregional con soporte horario automático.
  • Módulo de video-presentación para evaluación de habilidades comunicativas.
  • Pruebas de nivel de inglés integradas para filtrado de candidatos.
  • Integración nativa con Zoom para agendamiento y creación de entrevistas.
  • Componente API para sincronización bidireccional con el ERP de la empresa.
  • Panel administrativo centralizado para el equipo de HR y coordinadores.

// La Solución

Arquitectura & Enfoque Técnico

Se implementó una arquitectura monolítica utilizando Laravel como motor principal y Vue.js 2 para las capas de interacción reactiva. Este enfoque permitió un desarrollo ágil y una gestión de estado coherente para procesos complejos como el agendamiento de citas.

  • Flujo de Postulación: Los candidatos cargan su perfil y un video "Demo" que los reclutadores pueden calificar directamente en la plataforma.
  • Automatización de Entrevistas: Mediante la API de Zoom, el sistema genera automáticamente las salas de reunión y gestiona los enlaces de acceso tanto para el docente como para el entrevistador.
  • Conectividad ERP: Aunque la base es monolítica, desarrollé un componente de servicios API específico que permite que los datos de los candidatos seleccionados se integren sin fricción en el sistema de gestión de recursos humanos de la compañía.
  • Evaluación Académica: Motor de tests integrado para determinar el nivel de inglés del aplicante de forma inmediata tras el registro.
